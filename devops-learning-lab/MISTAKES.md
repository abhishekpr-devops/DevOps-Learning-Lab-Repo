# Mistakes Log

## Mistake 1
Started with wrong Git identity and GitHub showed wrong contributor.

### Fix
Changed author details and updated commit history.

### Lesson
Always check `git config user.name` and `git config user.email` before first push.

## Mistake 2
Used incomplete terminal command ending with `&& \`.

### Fix
Cancelled command and reran a corrected version.

### Lesson
Check the full command before pressing Enter.

## Mistake 3
Forgot that `tree` does not show hidden files by default.

### Fix
Used `tree -a` to verify `.gitignore`.

### Lesson
Verification must be complete, not partial.

# IAM Notes

## What is IAM
IAM stands for Identity and Access Management. It is used to control access to AWS resources.

## Important concepts
### Users
People or applications that need AWS access.

### Roles
Temporary identities with permissions that can be assumed by services or users.

### Permissions
Rules that define what actions are allowed or denied.

### Policies
JSON documents that define permissions.

## Least privilege
Give only the minimum permissions required to do the job.

## My notes
IAM is critical for AWS security. Bad permissions can create major risk.

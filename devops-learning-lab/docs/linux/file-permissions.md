# File Permissions

## What are file permissions
File permissions control who can read, write, or execute a file.

## Three permission types
- read (r)
- write (w)
- execute (x)

## Three user groups
- owner
- group
- others

## Example
`rw-r--r--`

Meaning:
- owner can read and write
- group can read
- others can read

## Why permissions matter
Permissions help protect files from unwanted access or changes.

## Useful commands
- `ls -l` to view permissions
- `chmod` to change permissions
- `chown` to change file owner

## My notes
Permissions are basic Linux security. Wrong permissions can block access or create security risks.

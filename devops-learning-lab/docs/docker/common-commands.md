# Common Docker Commands

## Pull image
`docker pull nginx`

## Run container
`docker run -d -p 8080:80 nginx`

## List running containers
`docker ps`

## List all containers
`docker ps -a`

## Stop container
`docker stop <container_id>`

## Remove container
`docker rm <container_id>`

## List images
`docker images`

## Remove image
`docker rmi <image_id>`

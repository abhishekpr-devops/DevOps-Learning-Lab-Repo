# Docker Basics

## What is Docker
Docker is a platform used to package applications and their dependencies into containers.

## Why Docker is useful
- same app can run in different environments
- reduces "works on my machine" problems
- makes deployment easier
- helps with isolation

## Image vs Container
### Image
An image is a packaged blueprint.

### Container
A container is a running instance of an image.

## My notes
Docker helps run apps in a predictable way across systems.

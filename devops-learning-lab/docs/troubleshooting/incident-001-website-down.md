# Incident 001: Website Down

## Date
2026-04-10

## Service
demo-app.example.com

## Symptoms
- website not loading
- browser timeout
- users unable to access service

## Checks Performed
- ping test
- DNS lookup
- web server status check
- port check
- application process check

## Possible Causes
- server down
- nginx or apache stopped
- firewall blocking traffic
- DNS issue
- application crashed

## Temporary Resolution
- restarted web service
- verified application port
- checked logs
- confirmed site loading again

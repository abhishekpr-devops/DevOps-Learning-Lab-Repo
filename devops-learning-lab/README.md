# DevOps Learning Lab

## Project Overview
This repository is my personal DevOps learning lab. I am using it to practice Git, Linux, Docker, AWS, troubleshooting, and project documentation in a structured way.

The goal is not just to collect notes. The goal is to learn by building, editing, making mistakes, fixing them, and tracking everything properly.

## Purpose
This repo helps me practice:

- writing clean documentation
- working with Git branches
- resolving merge conflicts
- using revert safely
- organizing technical notes
- thinking like a DevOps engineer

## Topics Covered
- Git basics
- merge conflicts
- revert vs reset
- Linux permissions
- systemctl basics
- useful Linux commands
- Docker basics
- common Docker commands
- AWS EC2 basics
- IAM notes
- troubleshooting workflow
- debugging checklists
